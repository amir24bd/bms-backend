
# Project Name: Blood Management System

### I have used django and Django Rest Framework (DRF) in the backend of my project. There is no need to run XAMPP and virtualenv to run the code of the backend part. Just install the project and app described below and 'python manage.py runserver' to run it.

## Backend have installed
# 1. pip install django
# 2. django-admin startproject bms_backend
# 3. django-admin stratapp core
# 4. pip install djangorestframework
# 5. pip install djangorestframework-simplejwt
# 6. pip install django-cors-headers

### I have used React and bootstrap 5 in the frontend of my project. After installing the apps mentioned below, you just need to run it. Please run the backend code and frontend code together.

## Frontend have installed
# 1. npx create-react-app bms-frontend
# 2. npm install axios react-router-dom bootstrap
